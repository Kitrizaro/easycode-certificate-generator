# EasyCode Certificate Generator

Одностраничный генератор сертификатов о поступлении в школу EasyCode.

## Что умеет

- Ввод ФИО ребёнка
- Ввод направления
- Ввод даты поступления
- Ввод преподавателя ПУ
- Экспорт сертификата в PDF
- Печать / сохранение через браузер
- Работает как обычный статический сайт на GitHub Pages

## Как запустить локально

Откройте файл `index.html` в браузере.

Для кнопки **«Скачать PDF»** нужен интернет, потому что HTML подключает библиотеки `html2canvas` и `jsPDF` через CDN. Если интернет недоступен, используйте кнопку **«Печать / сохранить как PDF»**.

## Как опубликовать на GitHub Pages

1. Создайте новый репозиторий на GitHub.
2. Загрузите в корень репозитория файлы из этого архива:
   - `index.html`
   - `.nojekyll`
   - `README.md`
3. Откройте **Settings → Pages**.
4. В блоке **Build and deployment** выберите:
   - **Source:** `Deploy from a branch`
   - **Branch:** `main`
   - **Folder:** `/ (root)`
5. Нажмите **Save**.
6. Через пару минут сайт будет доступен по адресу вида:

```text
https://ВАШ_ЛОГИН.github.io/НАЗВАНИЕ_РЕПОЗИТОРИЯ/
```

## Важно

Не переименовывайте `index.html`, иначе GitHub Pages не откроет сайт как главную страницу автоматически.
